═══════════════════════════════════════════════════════════════
📱 ABOUT THE APP
═══════════════════════════════════════════════════════════════

Aura App is a live streaming platform that allows users to:
• 📹 Broadcast live video to viewers
• 👥 Connect with other streamers
• 💬 Send direct messages
• 🎨 Customize profiles
• 🎮 Earn rewards (Aura Coins)
• 👑 Secret admin control (owner only)

═══════════════════════════════════════════════════════════════
✨ KEY FEATURES
═══════════════════════════════════════════════════════════════

🎥 LIVE STREAMING
• Start broadcasting with one tap
• Front camera (selfie) + back camera switching
• Real-time camera toggle (like TikTok)
• Microphone on/off control
• Live viewer count display

👤 PROFILE CUSTOMIZATION
• Change username
• Customize with colors (Purple, Blue, Pink, Green)
• Select profile frames (Classic, Gold, Diamond, Dragon)
• View follower stats
• Profile settings

🔐 SECURE AUTHENTICATION
• Firebase email/password auth
• Session persistence
• Secure logout

👑 SECRET ADMIN PANEL
• Unlock by tapping profile picture 5 times
• Manage users
• View analytics
• Ban users
• Distribute Aura Coins

═══════════════════════════════════════════════════════════════
🚀 QUICK START
═══════════════════════════════════════════════════════════════

1. Install APK on Android device
2. Grant camera & mic permissions
3. Create account
4. Go Live tab → Tap GO LIVE
5. Start streaming!

ADMIN ACCESS (Owner):
1. Profile tab (👤)
2. Tap profile picture 5 times quickly
3. Admin panel appears!
